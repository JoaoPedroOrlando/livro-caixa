Política de Privacidade – Livro Caixa

Última atualização: 28/06/2025

Esta Política de Privacidade descreve como o aplicativo Livro Caixa coleta, usa, armazena e compartilha suas informações.

1. Informações que Coletamos

O app coleta dados financeiros inseridos manualmente. Nenhum dado pessoal é coletado.

2. Armazenamento

Os dados ficam localmente no seu dispositivo.

3. Uso dos Dados

Usamos os dados apenas para exibir informações no próprio app e exportações se solicitadas.

4. Compartilhamento

Não compartilhamos seus dados com terceiros.

5. Segurança

Os dados estão no seu dispositivo. Recomendamos uso de senha no aparelho.

6. Direitos do Usuário

Você pode editar ou apagar suas informações a qualquer momento.

7. Alterações

Esta política pode mudar. Publicaremos aqui qualquer alteração relevante.
